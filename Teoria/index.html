Estrutura básica de um documento HTML:

<html>

<!DOCTYPE html>

<head>

   <meta charset="pt-br">

   <meta name="description" content="Descrição sobre o projeto">

   <title> Titulo </title>

</head>

<body>

   <header>...</header>

   <main>

      <section>...</section>

   </main>

   <footer>...</footer>

</body>

</html>

Vamos entender cada parte:

  • !DOCTYPE html: Declara o tipo de documento, informando ao navegador que se trata de um documento HTML5.
  • html lang="pt-br": Define a raiz do documento HTML e especifica o idioma do conteúdo como português do Brasil.
  • head: Contém metadados da página, como o título, codificação de caracteres e informações para dispositivos móveis.
  • meta charset="UTF-8": Especifica a codificação de caracteres utilizada na página.
  • title: Define o título da página, que aparece na aba do navegador.
  • body: Contém o conteúdo visível da página, como texto, imagens, links, etc.
  • header: Representa o cabeçalho da página, geralmente contendo o título, logotipo, menu de navegação, informações de contato e outros elementos introdutórios.
  • main: Contém o conteúdo principal da página, ou seja, o conteúdo mais relevante e específico daquela página.
  • footer: Representa o rodapé da página, geralmente contendo informações adicionais como copyright, links para políticas de privacidade, informações de contato, mapa do site, etc.

Principais tags HTML e suas funções:

  • <h1> até <h6>: Definem cabeçalhos de diferentes níveis, do mais importante (h1) ao menos importante (h6).
  • <a>: Cria um link para outra página ou recurso.
  • <img>: Insere uma imagem.
  • <ul> <ol>: Criam listas não ordenadas (ul) e ordenadas (ol), respectivamente.
  • <li>: Define um item dentro de uma lista.
  • <table>:Cria uma tabela.
  • <tr>: Define uma linha em uma tabela.
  • <td>: Define uma célula em uma tabela.
  • <div>: Define uma divisão, usada para agrupar elementos e aplicar estilos.
  • <p>: Define um parágrafo.
  • <span>: Define um trecho de texto inline, usado para aplicar estilos a partes específicas do texto.
  • <b>: Indica texto em negrito sem conotação semântica. Ou seja, o navegador irá exibir o texto em negrito, mas não há uma implicação de que o texto seja mais importante ou urgente.
  • <strong>: Além de tornar o texto em negrito, ela indica que o conteúdo é importante para o entendimento do texto. É mais semântica e pode ser utilizada por tecnologias assistivas para dar mais destaque a esse conteúdo.
  • <i>: Indica texto em itálico sem conotação semântica. É frequentemente usado para títulos de obras, termos técnicos, ou para indicar pensamento.
  • <em> Indica ênfase em um texto, mas de uma forma mais sutil que. É frequentemente usado para indicar palavras ou frases que são importantes para o significado de uma frase.
  • <br>: Utilizada para inserir uma quebra de linha em um parágrafo. Ela não fecha o parágrafo, apenas força o texto a quebrar para a próxima linha.
  • <abbr>: Em HTML é utilizada para definir uma abreviação ou acrônimo e, opcionalmente, fornecer uma descrição completa para ela. É especialmente útil para siglas, abreviações comuns ou termos técnicos que podem não ser familiares a todos os leitores.
